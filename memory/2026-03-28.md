### 4. OpenClaw cron vs 系统 cron（重要区别）

| | 系统 cron | OpenClaw cron |
|--|-----------|---------------|
| 管理方式 | `crontab -e` | `openclaw cron` |
| 驱动 | cron 守护进程 | OpenClaw 内部调度 |
| 依赖 | 系统独立 | 依赖 OpenClaw 进程 |
| 可靠性 | 高（系统级） | 中 |
| 任务隔离 | 完全隔离 | 共享会话 |
