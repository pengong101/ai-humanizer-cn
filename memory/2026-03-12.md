## 💡 经验教训

### 配置修改原则

1. **先备份** - 任何修改前先备份
2. **验证兼容性** - 确认 OpenClaw 版本支持
3. **仅使用文档化字段** - 不添加实验性配置
4. **修改后验证** - JSON 格式 + 必要字段
5. **权限保护** - 配置文件设置只读

### 更新流程

1. **备份优先** - 配置 + 记忆文件
2. **验证恢复** - 确保备份可用
3. **执行更新** - 拉取镜像 + 重启
