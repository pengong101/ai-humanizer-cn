### 优化方案

创建智能代理控制器：
- 脚本：clash-smart-controller.py
- 功能：根据查询内容自动开启/关闭代理
- 目标：节约 50-70% 流量费用

### 三种模式

1. 学术搜索模式
   - 触发：paper、论文、research、IEEE 等
   - 行为：启用代理
   - 场景：学术论文、专利检索

2. 专业情报模式
   - 触发：market、市场、analysis、报告等
   - 行为：启用代理
   - 场景：市场调研、竞争情报

3. 日常使用模式
   - 触发：其他所有查询
   - 行为：关闭代理
   - 场景：日常问答、简单查询

### 使用方式

手动控制：
```bash
python3 clash-smart-controller.py status   # 查看状态
python3 clash-smart-controller.py enable   # 启用代理
python3 clash-smart-controller.py disable  # 关闭代理
python3 clash-smart-controller.py stats    # 查看统计
```

自动控制：
```bash
python3 clash-smart-controller.py auto --query "毫米波雷达 学术论文"
# → 识别为学术 → 启用代理

python3 clash-smart-controller.py auto --query "今天天气"
# → 识别为日常 → 关闭代理
```

预计节约：50-70% 流量费用
