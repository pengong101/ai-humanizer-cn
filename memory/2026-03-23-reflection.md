## 🔍 根本原因分析

### 1. 图片管理流程问题

**问题表现：**
- 使用相同 Unsplash 关键词导致图片重复
- 未建立图片唯一性验证机制
- 下载后未验证图片内容

**根本原因：**
- 缺少图片选择规范
- 缺少 md5 去重检查
- 缺少图片内容验证步骤

**改进方案：**
```
图片选择流程：
1. 确定图片主题（与文章强相关）
2. 使用不同关键词搜索
3. 下载后立即验证（file 命令）
4. md5 去重检查
5. 人工确认图片内容
```

---

### 2. QC 审核机制缺失

**问题表现：**
- 玻璃瓶图片未被发现
- 0 字节文件未被清理
- 重复图片未被检测

**根本原因：**
- QC 智能体配置错误（使用文本模型检查图片）
- 缺少图片内容验证步骤
- 缺少最终人工审核

**改进方案：**
```
QC 图片检查清单：
1. ✅ 文件大小检查（>10KB）
2. ✅ md5 去重检查
3. ✅ file 命令验证类型
4. ✅ 人工确认图片内容
5. ✅ 与文章主题匹配度
```

---

### 3. 并发控制问题

**问题表现：**
- 同时下载多张图片导致 0 字节
- 文件写入冲突
- 打包时文件丢失

**根本原因：**
- 多个 wget 同时执行
- 未等待下载完成就验证
- 未使用锁机制

**改进方案：**
```
下载流程优化：
1. 串行下载（一次一张）
2. 下载后 sleep 3 秒
3. 验证文件大小
4. 验证文件类型
5. 确认成功后再下载下一张
```

---

### 4. 备用方案不足

**问题表现：**
- EHT 官网 403 后无备选
- NASA 图片链接失效
- 多次尝试相同 URL

**根本原因：**
- 未建立科学图片资源库
- 未预先测试图片链接
- 缺少降级方案

**改进方案：**
```
科学图片资源优先级：
1. ESA Hubble (cdn.esahubble.org) ⭐⭐⭐⭐⭐
2. ESO 官网 (eso.org) ⭐⭐⭐⭐⭐
```
