## 🛠️ 问题与解决方案

### 问题 1：Docker 端口映射失败

**症状：** SearXNG 容器端口 8081 在宿主机未监听

**原因：** Docker-in-Docker 环境限制
- 容器内无 iptables 权限
- 无法修改宿主机网络规则
- 端口映射需要宿主机级别权限

**解决方案：** 使用容器 IP 直连
```python
SEARXNG_URL = "http://172.17.0.3:8080"  # 容器 IP 直连
```

**状态：** ✅ 已解决，搜索功能 100% 可用

---

### 问题 2：SearXNG JSON API 403

**症状：** JSON API 返回 403 Forbidden

**原因：** SearXNG 默认配置未启用 JSON 格式

**解决方案：**
1. 短期：使用 HTML 解析（已实现）
2. 中期：挂载自定义 settings.yml（需要宿主机操作）
3. 长期：使用容器 IP 直连（已实现，JSON API 可用）

**状态：** ✅ 已解决
