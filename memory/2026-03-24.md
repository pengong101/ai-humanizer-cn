### 解决方案（已实现）

#### 1. 组合关键词策略

**原则：** `具体主题 + 官方来源 + 图片类型`

**示例：**
```
❌ 旧：black hole（宽泛）
✅ 新：M87 black hole EHT official（精准）

❌ 旧：exoplanet
✅ 新：Kepler-452b NASA artist concept
```

**每张图片 3 组关键词：**
- 主关键词：最优组合
- 备选 1：同义词替换
- 备选 2：不同机构来源

---

#### 2. 多模态内容验证

**流程：**
```
下载图片
    ↓
qwen3.5-plus 多模态识别
    ↓
匹配度评分 (0-10 分)
    ↓
≥7 分 → 通过 ✅
<7 分 → 重试（最多 3 次）
```

**验证 Prompt：**
```
请识别这张图片的内容：
1. 图片中主要展示的是什么？
2. 是否有文字标注？
3. 图片风格是什么？
4. 是否与描述匹配：[预期描述]

输出：匹配度评分 + 理由 + 建议
```
