## 🔧 自动化工具

### GitHub 搜索脚本

```bash
#!/bin/bash
# search-github.sh

KEYWORD=$1
MIN_STARS=$2

curl -s "https://api.github.com/search/repositories?q=$KEYWORD+stars:>$MIN_STARS&sort=stars&order=desc" | \
  jq -r '.items[] | "\(.name): \(.stargazers_count) ⭐ - \(.html_url)"'
```

**使用：**
```bash
./search-github.sh "proxy+detection" 100
```

---

### ClawHub 搜索脚本

```bash
#!/bin/bash
# search-clawhub.sh

KEYWORD=$1
clawhub search "$KEYWORD" | head -20
```

**使用：**
```bash
./search-clawhub.sh "browser automation"
```
