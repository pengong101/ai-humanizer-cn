# Tavily Search - 精准搜索利器

Tavily AI 驱动的精准搜索，专注深度研究和事实核查。

---

## 🎯 使用场景

### 场景 1：深度技术研究

```
需求："调研最新的 LLM 推理优化技术"
     ↓
Tavily Search: 精准搜索学术论文、技术博客
     ↓
输出：10 篇高质量文献 + 关键发现摘要
```

---

### 场景 2：竞品分析

```
需求："分析 OpenClaw 竞品情况"
     ↓
Tavily Search: 搜索竞品信息、用户反馈、技术对比
     ↓
输出：竞品分析报告 + SWOT 分析
```

---

### 场景 3：事实核查

```
需求："验证某个技术声明的准确性"
     ↓
Tavily Search: 搜索多个可靠来源
     ↓
输出：验证结果 + 来源引用
```

---

## 🔧 核心功能

### 1. 精准搜索

**特点：**
- AI 理解查询意图
- 过滤低质量结果
- 优先权威来源
- 支持复杂查询

---

### 2. 结果提炼

**输出格式：**
```markdown
# 搜索结果摘要

## 关键发现
1. 发现 1（来源：xxx）
2. 发现 2（来源：xxx）

## 相关文献
- [标题](链接) - 来源
- [标题](链接) - 来源

## 置信度评分
- 高置信度：80%
- 中置信度：15%
- 低置信度：5%
```

---

### 3. 引用管理

**功能：**
- 自动记录来源
- 生成引用格式
- 验证链接有效性
- 存档网页快照
