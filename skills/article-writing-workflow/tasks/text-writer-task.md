# TextWriter 任务提示词

## 输入
- 选题：[文章主题]
- 受众：[目标读者]
- 字数：1500-3000 字
- 平台：微信公众号

## 写作要求

### 禁止写入
- "首先""其次""最后""综上所述"（用 ai-humanizer-cn 统一处理）
- 未注明来源的科学声称
- 政治敏感内容
- 个人隐私信息

### 必要元素
- 摘要（50-80字，放在文章开头）
- 小标题（## 级别，每篇 3-4 个）
- 具体数据或案例
- 术语首次出现时加括号解释

### 风格
- 口语化，像和读者聊天
- 段落不超过4句
- 每800字一段较佳

## 输出
```markdown
# 文章标题

> 摘要（50-80字）

---

## 小标题1

正文内容...

## 小标题2

正文内容...

---

*参考资料：来源1、来源2*
```

## 自检报告（必须同时输出）
```markdown
# 自检报告

## 字数统计
总字数：XXX 字

## 套话检测
[ ] 无"首先/其次/最后"序列

## 逻辑自检
[ ] 有具体数据支撑
[ ] 无未经证实声称

## 原创度
[ ] 无段落重复
```
