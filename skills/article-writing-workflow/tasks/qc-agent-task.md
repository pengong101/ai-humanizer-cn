# QCAgent v2.0 任务提示词

## 输入
- 完整文章（含图片Markdown）
- 选题和受众

## 四维评分体系

每项 0-10，总分 40 分

### 1. 合规安全（权重 25%）

检查项：
- [ ] 无政治敏感内容
- [ ] 无未经证实科学声称
- [ ] 无个人隐私信息泄露
- [ ] 图片版权合规（CC/Official/AI生成）
- [ ] 无抄袭/过度引用
- [ ] 事实陈述有来源

扣分项（每项 -1~2）：
- 单项事实错误
- 来源不明的数据
- 图片版权不明确

**危险信号（直接拒绝）：**
- 合规项 < 4 分 → 总分直接判 < 24

---

### 2. 文本质量（权重 30%）

检查项：
- [ ] 阅读流畅，无语法错误
- [ ] 无"首先/其次/最后"生硬序列
- [ ] 无重复段落或车轱辘话
- [ ] 术语解释充分
- [ ] 有具体案例或数据
- [ ] AI 痕迹已去除（通过 ai-humanizer-cn）

扣分项：
- 重复内容：-2/段
- 术语无解释：-1/次
- 逻辑跳跃：-2/处

---

### 3. 图片质量（权重 25%）

检查项：
- [ ] 每张图片与内容相关
- [ ] alt 文字描述准确
- [ ] 版权信息已标注
- [ ] 清晰度适合手机端展示
- [ ] 封面图有吸引力
- [ ] 无损坏图片链接

扣分项：
- 图片与内容不相关：-3/张
- alt 文字缺失：-1/张
- 版权不明：-2/张

---

### 4. 格式正确（权重 20%）

检查项：
- [ ] 字数在 1500-3000 范围内
- [ ] 标题层级清晰（## / ###）
- [ ] 微信公众号可直接粘贴
- [ ] 引用块/加粗正确使用
- [ ] 无乱码或格式错误

扣分项：
- 字数偏差 > 500：-2
- 格式导致公众号排版混乱：-3

---

## 评分输出格式

```markdown
# QC 报告 - [文章标题]
日期：[YYYY-MM-DD]

## 四维评分

| 维度 | 得分 | 状态 |
|------|------|------|
| 合规安全 | X/10 | ✅/🔄/❌ |
| 文本质量 | X/10 | ✅/🔄/❌ |
| 图片质量 | X/10 | ✅/🔄/❌ |
| 格式正确 | X/10 | ✅/🔄/❌ |
| **总分** | **XX/40** | **✅/🔄/❌** |

## 问题列表

### 🔴 严重问题（必须修复）
1. [具体问题] → [修改建议]

### 🟡 一般问题（建议修复）
1. [具体问题] → [修改建议]

## 修改指令（发送给对应智能体）
- **TextWriter**：[具体修改要求]
- **ImageAgent**：[具体修改要求]
- **Integrator**：[具体修改要求]

## 最终判定
✅ **通过**（总分≥32，各维≥6）
🔄 **修改后重审**（总分24-31，或某维<6）
❌ **拒绝重做**（总分<24或合规<4）
```

## 修改轮次规则
- 第1轮不合格 → 返回对应智能体修改
- 第2轮不合格 → 返回写作智能体精简/深化
- 第3轮仍不合格 → 建议重新选题
