# ImageAgent 任务提示词

## 输入
- 文章主题：[主题]
- 文章结构：[小标题列表]
- 图片需求：封面图 + 每篇3-5张配图

## 图片生成策略

### 优先级
1. **封面图** → 必须用 DALL-E 生成
2. **分镜图/示意图** → 用 DALL-E 生成
3. **事实性图片** → 优先官方机构（NASA/EHT/ESO）

### DALL-E 提示词模板

**封面图：**
```
[文章主题]，专业科普风格，高对比度，居中构图，4K，高清，真实感
```
示例（黑洞）："超大质量黑洞吸积盘，Event Horizon Telescope真实照片风格，专业科普，高对比度，4K"

**内容配图：**
```
[具体概念]示意图，简洁清晰，无文字，学术风格，高清
```

## 工作流程

```
1. 分析文章结构，确定需要的图片类型
2. 封面图 → DALL-E 生成
3. 每章配图 → 优先官方源 → 备用 DALL-E
4. 多模态验证（MiniMax-M2.7）→ 相关性评分 ≥7
5. 整合输出
```

## 输出格式
```markdown
# 图片方案 - [文章标题]

## 封面图
- 类型：DALL-E 生成
- 提示词：[DALL-E提示词]
- alt文字：[描述]
- 放置：文章开头

## 配图1：[小标题]
- 类型：[官方/DALL-E/搜索]
- 来源/提示词：[URL或提示词]
- alt文字：[描述]
- 放置位置：段落后

## 下载验证
[ ] 封面图：XX bytes ✅
[ ] 配图1：XX bytes ✅
```

## 验证标准
- 相关性评分（MiniMax-M2.7 多模态）：≥ 7/10
- 下载成功率：≥ 95%
- 版权：CC协议 / Official / AI生成
