# Find Skills - 技能导航仪

帮你找到合适的技能，或者发现需要开发的新技能。

---

## 🎯 使用场景

### 场景 1：发现技能需求

```
任务：需要自动化处理 Excel 报表
     ↓
Find Skills: 分析需求 → 搜索现有技能
     ↓
结果：发现 Office-Automation 技能可用
     ↓
建议：安装 Office-Automation 技能
```

---

### 场景 2：评估技能价值

```
需求：需要一个新技能来做 X
     ↓
Find Skills: 评估价值（使用频率、开发成本、替代方案）
     ↓
建议：
- 高价值 → 开发新技能
- 中价值 → 寻找替代
- 低价值 → 手动处理
```

---

### 场景 3：技能库管理

```
定期扫描技能库
     ↓
Find Skills: 分析使用情况
     ↓
报告：
- 热门技能（继续使用）
- 冷门技能（考虑废弃）
- 缺失技能（建议开发）
```

---

## 🔧 核心功能

### 1. 需求分析

**分析维度：**
- 任务类型
- 使用频率
- 复杂度
- 替代方案

**输出：**
- 技能需求描述
- 优先级评估
- 开发/安装建议

---

### 2. 技能搜索

**搜索范围：**
- 本地技能库
- ClawHub 市场
- GitHub 项目
- 社区推荐

**匹配度评估：**
- 功能匹配度（%）
- 质量评分（⭐）
- 维护状态（活跃/停滞）

---

### 3. 价值评估

**评估维度：**
```
价值 = 使用频率 × 节省时间 × 质量提升
成本 = 开发时间 + 维护成本 + 学习成本

ROI = 价值 / 成本
```

**决策阈值：**
- ROI > 3 → 强烈推荐开发
- ROI 1-3 → 考虑开发
- ROI < 1 → 不推荐

---

## 📋 使用流程

### 步骤 1：描述需求

```markdown
任务描述：需要自动化处理每周销售报表
当前方式：手动从数据库导出，Excel 处理，耗时 2 小时/周
期望效果：自动化生成，耗时<10 分钟/周
使用频率：每周 1 次
```

---

### 步骤 2：技能分析

**Find Skills 分析：**
```
任务类型：数据处理 + 报告生成
复杂度：中等
替代方案：
  - Office-Automation 技能（匹配度 80%）
  - 自定义脚本（开发时间 4 小时）
  - 继续手动（每周 2 小时）
```

---

### 步骤 3：建议输出

**推荐方案：**
```
方案 1（推荐）：使用 Office-Automation 技能
  - 匹配度：80%
  - 学习成本：1 小时
  - 节省时间：1.5 小时/周
  - ROI: 高

方案 2：自定义脚本
  - 开发时间：4 小时
  - 维护成本：低
  - 节省时间：1.8 小时/周
  - ROI: 中

方案 3：继续手动
  - 成本：2 小时/周
  - ROI: 低
```

## 📊 技能库管理

### 技能注册表

**位置：** `/root/.openclaw/workspace/skills/registry.json`

**内容：**
```json
{
  "skills": [
    {
      "name": "ai-humanizer-cn",
      "category": "content",
      "owner": "COO",
      "status": "active",
      "usage_count": 150,
      "last_used": "2026-03-11"
    }
  ]
}
```

---

### 定期扫描

**频率：** 每周一次

**扫描内容：**
- 技能使用情况
- 技能文档更新
- 技能依赖检查
- 新技能需求发现

**输出：** 技能库健康报告

## 🎯 最佳实践

### ✅ 应该做的

- 定期扫描技能库
- 记录技能使用情况
- 及时更新技能文档
- 分享技能发现
- 评估技能 ROI

### ❌ 不应该做的

- 重复开发相同技能
- 使用未注册技能
- 不记录技能使用
- 忽略技能维护
- 过度依赖单一技能
