# Multi Search Engine - 全网搜索聚合器

一次搜索，全网结果。自动聚合、去重、排序。

---

## 🎯 使用场景

### 场景 1：全面信息收集

```
查询："OpenClaw AI Assistant"
     ↓
Multi Search: 同时搜索 10+ 平台
     ↓
输出：聚合结果（去重 + 排序）
```

---

### 场景 2：跨平台对比

```
查询："最佳 AI 写作工具"
     ↓
Multi Search: 搜索知乎、小红书、公众号、GitHub
     ↓
输出：各平台观点对比
```
