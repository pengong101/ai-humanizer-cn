# 热点选题工作流 v3.0

**日期：** 2026-04-03  
**版本：** v3.0  
**定位：** 内容创作的前置工作流

---

## 一、流程概览

```
用户需求（科普文章选题）
    │
    ▼
main（布置任务）
    │
    ▼
ResearchAgent 搜集热点
    │
    ├─ 搜索多平台热点
    ├─ 聚合整理
    └─ 初筛候选
    │
    ▼
生成候选题目列表
    │
    ▼
用户选择题目
    │
    ▼
进入内容创作工作流（WORKFLOW-CONTENT-V3.md）
```

---

## 二、详细流程

### 阶段1：任务布置（main）

```markdown
## 科普文章选题任务
**日期：** [YYYY-MM-DD]
**方向：** [如：AI科技 / 太空探索 / 生物医学]
**数量要求：** [如：3-5个候选]
**特殊要求：** [如：近期热点、有数据支撑]

进入 ResearchAgent 热点搜集模式...
```

---

### 阶段2：热点搜集（ResearchAgent）

ResearchAgent 使用 Agent Reach 跨平台搜集热点：

```bash
# 1. 全网热点搜索
mcporter call exa.web_search_exa query="[方向] 科普 热门 突破 2026" numResults=15 freshness=month

# 2. 公众号热点
mcporter call exa.web_search_exa query="[方向] site:mp.weixin.qq.com 科普 热门" numResults=10

# 3. 知乎热点
mcporter call exa.web_search_exa query="[方向] site:zhihu.com 科普 热门" numResults=10

# 4. B站/视频内容
mcporter call exa.web_search_exa query="[方向] B站 科普 热门视频" numResults=5
```

## 七、技术实现

### Agent Reach 搜索脚本

```bash
#!/bin/bash
# topic-search.sh - 热点选题搜索

DIRECTION=$1  # 如 "AI科技"
NUM_RESULTS=${2:-10}  # 默认10条

echo "=== 全网热点搜索 ==="
mcporter call exa.web_search_exa query="$DIRECTION 科普 热门 突破 2026" numResults=$NUM_RESULTS freshness=month

echo "=== 公众号热点 ==="
mcporter call exa.web_search_exa query="$DIRECTION site:mp.weixin.qq.com 科普" numResults=5

echo "=== 知乎热点 ==="
mcporter call exa.web_search_exa query="$DIRECTION site:zhihu.com 科普 热门" numResults=5

echo "=== B站视频 ==="
mcporter call exa.web_search_exa query="$DIRECTION B站 科普 热门" numResults=5
```

### 使用方式

```bash
# 搜索 AI 科技热点
bash topic-search.sh "AI科技"

# 搜索 太空探索热点
bash topic-search.sh "太空探索"

# 自定义数量
bash topic-search.sh "AI科技" 15
```
