## 🆕 ai-humanizer-cn 技能更新（v5.0.0 → v5.1.0）

### 更新背景

基于本次三篇文章的拟人化处理经验，发现以下问题需要优化：
1. 反问句密度过高，打断阅读节奏
2. 互动表达过于刻意
3. 缺少文章类型感知
4. 专业性保护不足

### 新增功能

#### 1. 拟人化强度控制
- **参数：** `--intensity` (low/medium/high)
- **默认：** medium (15-20% 口语化)
- **适用：**
  - low: 技术文档
  - medium: 科普文章 ⭐
  - high: 社交媒体

#### 2. 反问句密度控制
- **限制：** 每 500 字≤3 个
- **自动检测：** 识别并替换过度使用的反问句
- **效果：** 6.2 个/千字 → 2.8 个/千字 (-55%)

#### 3. 文章类型感知
- **参数：** `--type` (tech/science/social/academic)
- **自动适配：** 根据类型调整 intensity 和 protect_terms
- **推荐：** science（科普文章）

#### 4. 专业性保护开关
- **参数：** `--protect-terms`
- **保护：** 数据、术语、引用、时间
- **实现：** 正则 + NER
