## ✅ 解决方案

### 一、组合关键词策略

**原则：** `具体主题 + 官方来源 + 图片类型` 三要素组合

**示例对比：**

| 文章主题 | ❌ 旧方法（宽泛） | ✅ 新方法（组合） |
|----------|-----------------|------------------|
| 黑洞照片 | `black hole` | `M87 black hole EHT official` |
| 系外行星 | `exoplanet` | `Kepler-452b NASA artist concept` |
| 6G 通信 | `6G network` | `terahertz wave electromagnetic spectrum diagram` |

**每张图片生成 3 组关键词：**
- 主关键词：最优组合
- 备选 1：同义词替换
- 备选 2：不同机构来源

---

### 二、下载后多模态验证

**流程：**
```
下载图片
    ↓
qwen3.5-plus 多模态识别
    ↓
识别结果 vs 预期关键词
    ↓
匹配度评分 (0-10 分)
    ↓
┌────┴────┐
│ ≥7 分？  │
└────┬────┘
   ┌─┴─┐
   ↓   ↓
  是   否
   ↓   ↓
通过  重新查找
✅   (使用备选关键词)
```

**验证 Prompt：**
```
请识别这张图片的内容：

1. 图片中主要展示的是什么？
2. 是否有文字标注？内容是什么？
3. 图片风格是什么？（照片/艺术图/示意图/光谱图）
4. 是否与以下描述匹配：[预期关键词描述]

请输出：
- 匹配度评分：0-10 分
- 匹配理由：简要说明
- 建议：通过/重新查找
```

## 📝 使用示例

### 示例：黑洞文章图片选取

**步骤 1：生成图片需求清单**

```json
{
  "article_theme": "黑洞照片",
  "images": [
    {
      "position": "封面",
      "primary": "M87 black hole EHT official",
      "alternatives": [
        "M87* event horizon telescope",
        "first black hole image 2019"
      ],
      "description": "2019 年发布的 M87 黑洞照片，橙色圆环"
    }
  ]
}
```

**步骤 2：搜索并下载**

```bash
# 使用主关键词搜索
wget "https://eventhorizontelescope.org/sites/default/files/M87-black-hole-image.jpg" \
     -O /tmp/test-m87.jpg
```

**步骤 3：多模态验证**

发送图片给 qwen3.5-plus：
```
请识别这张图片的内容：

1. 图片中主要展示的是什么？
2. 是否有文字标注？内容是什么？
3. 图片风格是什么？
4. 是否与以下描述匹配：2019 年发布的 M87 黑洞照片，橙色圆环

请输出：
- 匹配度评分：0-10 分
- 匹配理由
- 建议：通过/重新查找
```

**步骤 4：根据评分决策**

- 评分 ≥ 7 → 保存图片到文章目录
- 评分 < 7 → 使用备选关键词重新搜索
