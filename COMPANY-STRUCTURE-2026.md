## 📋 执行摘要

**公司定位：** 科技科普内容创作 + 前沿技术研究 + 资本运作

**核心业务：**
1. 📺 科普内容制作（文章 + 视频）
2. 📱 多平台运营（抖音、小红书、B 站、微信公众号、视频号）
3. 🔬 深度研究（雷达、通信、信号处理、物理学、AI）
4. 📈 资本运作（A 股、港股、美股 ETF 中线/波段投资）

**组建原则：** 科学、高效、扁平化、结果导向

## 👥 核心岗位详细说明

### CEO（首席执行官）

**汇报对象：** 董事会/创始人  
**下属：** CTO、COO、CFO

**核心职责:**
- 制定公司战略和发展方向
- 融资和资源整合
- 重大决策和风险管理
- 公司文化和团队建设

**关键能力:**
- 🔭 **战略眼光** - 能看到 3-5 年后的趋势
- 💪 **执行力** - 决策果断，说到做到
- 🌐 **大局观** - 平衡短期和长期利益
- 🤝 **领导力** - 凝聚团队，激发潜能

**马斯克式特质:**
- 第一性原理思考
- 跨学科知识整合
- 极致追求（做到最好）
- 长期主义

---

### CTO（首席技术官）

**汇报对象：** CEO  
**下属：** 研究团队、技术团队

**核心职责:**
- 技术研究方向规划
- 研究团队管理和指导
- 技术内容质量把控
- 技术工具和方法论建设

**关键能力:**
- 📡 **专业背景** - 雷达/通信/信号处理/物理学/AI
- 📚 **研究能力** - 深度研究、论文阅读
- ✍️ **表达能力** - 将复杂技术通俗化
- 👨‍💼 **管理能力** - 团队管理、项目管理

**优先领域:**
1. 雷达技术（毫米波、相控阵）
2. 通信技术（5G/6G、卫星通信）
3. 信号处理
4. AI 应用

---

### COO（首席运营官）

**汇报对象：** CEO  
**下属：** 内容团队、运营团队

**核心职责:**
- 内容生产和质量把控
- 多平台运营策略
- 用户增长和品牌建设
- 商业化探索

**关键能力:**
- 📺 **内容敏感度** - 知道什么内容会火
- 📱 **平台运营** - 熟悉各平台规则
- 📊 **数据分析** - 用数据驱动决策
- 💼 **商业化** - 变现模式探索

**平台重点:**
1. 抖音（短视频）
2. 小红书（图文 + 视频）
3. B 站（长视频）
4. 微信公众号（深度文章）
5. 视频号（微信生态）

---

### CFO（首席财务官）

**汇报对象：** CEO  
**下属：** 投资团队、财务团队

**核心职责:**
- 投资策略制定和执行
- 财务管理和风险控制
- 融资和资本运作
- 税务筹划

**关键能力:**
- 📈 **投资能力** - 行业研究、价值判断
- 💰 **财务管理** - 预算、现金流管理
- ⚖️ **风险控制** - 识别和规避风险
- 📋 **合规意识** - 税务、法律合规

**投资方向:**
1. A 股（科技、制造）
2. 港股（互联网、科技）
3. 美股 ETF（科技、指数）
4. 行业：雷达、通信、AI、半导体

**投资周期：** 1-3 个月（波段） ~ 6 个月（中线）

## 🎯 招聘策略

### 招聘原则

1. **宁缺毋滥** - 只招最优秀的人
2. **能力优先** - 学历是参考，能力是关键
3. **文化匹配** - 认同公司价值观
4. **成长潜力** - 有学习能力和上进心

### 招聘渠道

| 渠道 | 适用岗位 | 成本 | 效果 |
|------|---------|------|------|
|  Boss 直聘 | 运营、制作 | 中 | ⭐⭐⭐⭐ |
|  拉勾网 | 技术、产品 | 中 | ⭐⭐⭐⭐ |
|  猎聘 | 高管、专家 | 高 | ⭐⭐⭐ |
|  内推 | 所有岗位 | 低 | ⭐⭐⭐⭐⭐ |
|  高校合作 | 研究员、实习 | 低 | ⭐⭐⭐ |

### 薪酬体系

**原则：** 高于市场平均，激励优秀

| 级别 | 月薪范围 | 股权激励 | 说明 |
|------|---------|---------|------|
| 高管 (C-level) | 50-100K | 5-10% | 核心决策层 |
| 总监 | 30-50K | 1-3% | 部门负责人 |
| 高级 | 20-30K | 0.5-1% | 骨干员工 |
| 中级 | 12-20K | - | 熟练员工 |
| 初级 | 8-12K | - | 新人/助理 |

## 💼 工作流程设计

### 内容生产流程

```
选题策划 → 资料收集 → 脚本撰写 → 审核 → 
拍摄 → 剪辑 → 审核 → 发布 → 数据分析 → 优化
```

### 研究工作流程

```
方向选择 → 文献调研 → 深入研究 → 
技术验证 → 内容转化 → 应用探索
```

### 投资决策流程

```
行业研究 → 标的筛选 → 深度分析 → 
风险评估 → 投资决策 → 投后管理 → 退出
```
