# 专家智能体 (Domain Expert Agent)

**版本：** v0.1  
**创建时间：** 2026-03-22  
**领域：** 毫米波雷达  
**状态：** P0 阶段实施中

---

## 一、智能体概述

### 1.1 职责

- 术语校验（检查技术术语使用是否准确）
- 参数核实（验证技术参数是否在合理范围）
- 趋势分析（基于专业知识判断行业发展方向）
- 内容增强（补充专业背景说明与技术对比）
- 问答支持（回答用户关于雷达技术的专业问题）

### 1.2 模型配置

| 任务类型 | 推荐模型 | 说明 |
|---------|---------|------|
| 专业分析 | qwen3-max | 复杂技术问题分析 |
| 知识检索 | qwen3.5-plus + RAG | 向量数据库检索 |
| 快速查询 | qwen3.5-flash | 简单问题快速响应 |

### 1.3 知识库

- 毫米波雷达技术论文
- 行业标准与规范
- 供应商技术文档
- 专利数据库

## 二、API 设计

### 2.1 术语校验

```python
POST /api/v1/expert/validate-terms
{
  "text": "文本内容",
  "domain": "mmwave_radar"
}
```

### 2.2 参数核实

```python
POST /api/v1/expert/verify-params
{
  "params": {
    "frequency": "77GHz",
    "range": "200m",
    "resolution": "0.1m"
  },
  "domain": "mmwave_radar"
}
```

### 2.3 技术问答

```python
POST /api/v1/expert/qa
{
  "question": "4D 成像雷达与传统毫米波雷达的区别？",
  "domain": "mmwave_radar"
}
```
