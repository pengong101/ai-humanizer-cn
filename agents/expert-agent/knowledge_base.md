## 一、术语库

### 1.1 基础术语

| 术语 | 英文 | 说明 |
|------|------|------|
| FMCW | Frequency Modulated Continuous Wave | 调频连续波，毫米波雷达常用调制方式 |
| MIMO | Multiple Input Multiple Output | 多输入多输出，增加虚拟通道数 |
| 4D 成像雷达 | 4D Imaging Radar | 增加高度维度的毫米波雷达 |
| 点云 | Point Cloud | 雷达检测到的目标点集合 |
| 波束成形 | Beamforming | 控制天线阵列信号方向 |
| 多普勒 | Doppler | 频率偏移，用于测速 |

### 1.2 技术参数

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 频率 | 24GHz / 77GHz | 车载雷达常用频段 |
| 带宽 | 1-4GHz | 决定距离分辨率 |
| 探测距离 | 0.5-300m | 短距/中距/长距 |
| 距离分辨率 | 0.1-0.5m | 区分两个目标的最小距离 |
| 速度分辨率 | 0.1-1m/s | 区分两个目标的最小速度差 |
| 角度分辨率 | 1-5 度 | 区分两个目标的最小角度差 |

## 三、行业趋势

### 3.1 4D 成像雷达

- **时间线：** 2024-2026 量产
- **特点：** 增加高度维度，点云密度接近激光雷达
- **优势：** 全天候工作，成本低于激光雷达
- **挑战：** 数据处理复杂度高

### 3.2 芯片集成化

- **趋势：** 单芯片集成收发器 + DSP
- **优势：** 降低成本、功耗、体积
- **厂商：** TI、Infineon、NXP

### 3.3 AI 融合

- **应用：** 目标识别、分类、跟踪
- **技术：** 深度学习、点云处理
- **时间线：** 2025+ 成为标配

## 四、常见问题

### Q1: 24GHz 和 77GHz 雷达的区别？

**A:** 
- 24GHz：带宽窄（250MHz），分辨率低，逐渐淘汰
- 77GHz：带宽宽（4GHz），分辨率高，车载主流

### Q2: 4D 成像雷达 vs 传统毫米波雷达？

**A:**
- 传统：3D（距离 + 速度 + 方位角）
- 4D: 增加高度维度，点云密度提升 10-100 倍

### Q3: 毫米波雷达 vs 激光雷达？

**A:**
- 毫米波：全天候（雨雾穿透），成本低，精度较低
- 激光雷达：高精度，高成本，受天气影响
