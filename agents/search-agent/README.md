# 搜索智能体 (Search Agent)

**版本：** v0.1  
**创建时间：** 2026-03-23  
**职责：** 信息搜集、多引擎搜索、结果聚合

---

## 一、核心职责

### 1.1 信息搜集
- 理解搜索意图
- 生成搜索关键词
- 执行多引擎搜索

### 1.2 多引擎搜索
- SearXNG（主引擎）
- 备用搜索引擎
- 专业数据库

### 1.3 结果聚合
- 去重
- 相关性排序
- 时效性过滤

---

## 二、API 设计

### 2.1 搜索

```python
POST /api/v1/search/query
{
  "query": "毫米波雷达 最新进展",
  "engines": ["baidu", "bing", "google"],
  "time_range": "week",
  "max_results": 20
}
```

### 2.2 聚合结果

```python
GET /api/v1/search/aggregate/{search_id}
```
