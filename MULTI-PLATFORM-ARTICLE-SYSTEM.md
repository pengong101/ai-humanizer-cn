## 📝 版本生成流程

### 流程说明

```
初稿完成 → 小红书版本 → 公众号版本 → 知乎版本
   ↓           ↓            ↓            ↓
 2500 字    1200 字      3000 字      4000 字
```

### 转换规则

**小红书版本:**
```python
def convert_to_xiaohongshu(article):
    # 1. 缩短字数 (2500→1200)
    content = shorten_content(article, 1200)
    
    # 2. 添加 emoji
    content = add_emoji(content)
    
    # 3. 缩短段落
    content = shorten_paragraphs(content)
    
    # 4. 添加标签
    tags = generate_tags(article)
    
    # 5. 生成配图建议
    images = generate_xiaohongshu_images(article)
    
    return {
        'content': content,
        'tags': tags,
        'images': images
    }
```

**公众号版本:**

def convert_to_wechat(article):
